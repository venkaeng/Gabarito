Lembretes e dados

✈️ Lembretes via Telegram

Receba notificações no seu Telegram para não esquecer de registrar. Siga os passos abaixo para configurar uma vez só.

  • Abra o Telegram e pesquise por @BotFather. Digite /newbot e siga as instruções para criar seu bot. Copie o Token que ele te dar.
  • Pesquise pelo seu novo bot no Telegram e aperte Iniciar. Depois acesse: api.telegram.org/bot[TOKEN]/getUpdates (substitua o token). Copie o número em "id" dentro de "chat".
  • Cole o Token e o Chat ID abaixo e salve.

💾 Backup e Restauração

Salve seus dados em um arquivo JSON para não perder nada se o navegador limpar o cache. Importe para fundir com registros de outro dispositivo — duplicatas são ignoradas automaticamente.

🗑️ Limpar dados

Remove todos os registros salvos neste dispositivo. Esta ação não pode ser desfeita.
